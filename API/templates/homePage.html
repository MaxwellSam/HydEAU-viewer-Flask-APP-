HydrEAUviewer API

Description

HydrEAUviewer API is an application programming interface for recovering hydrological data from hubEAU. HubEAU is also an API which allow a simple acess to data on water. Then, this API is an overlayer interface which simplify data recovery for the HydEAUviewer Web APP.

API Routes

Hydrological Stations: /hydro/stations

Elaborated data: /hydro/elab

Real Time data: /hydro/tr
